# Changelog

所有重要的版本更新都会记录在此。

## [v0.0.13] - 2026-04-20

### Added
-  添加多用户收藏管理功能，支持通过用户别名将歌曲添加到指定用户的收藏（如"爸爸收藏歌曲"）
### Fixed
- 修复了播放列表中文数字转换问题

## [v0.0.12] - 2026-04-14

### Add
- 添加多用户收藏播放功能

## [v0.0.11] - 2026-04-10

### Fixed
- 修复了收藏歌曲后没有继续播放的问题

### Added
- 增加收藏成功播报TTS

## [v0.0.10] - 2026-04-09

### Added
- 新增"收藏当前歌曲" / "添加当前歌曲到收藏"命令

## [v0.0.9] - 2026-04-08

### Fixed
- 修复重名歌曲因 key 相同无法正确获取的问题，将歌曲标识 key 改为 "歌曲名 - 专辑名 - 歌手" 格式

## [v0.0.8] - 2026-04-08

### Fixed
- 修复歌曲名中包含"的"规则匹配失败的问题
